## 🔹 프로젝트 버전

이 프로젝트는 두 가지 버전으로 구성되어 있습니다.

- **v0**: Python으로만 구현된 초기 버전
- **v2**: Python과 Spring Boot를 함께 사용하여 개선된 버전
