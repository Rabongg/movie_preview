# 이메일 디자인 가이드

## 레이아웃

- 극장별로 섹션을 나눠 그룹핑
- 각 섹션 내 이벤트 카드는 2열 그리드로 배치
- 최대 너비 600px

## 색상

### 극장 타이틀
| 극장 | 색상 |
|------|------|
| Megabox | `#3f51b5` (남색) |
| LotteCinema | `#e65100` (주황) |
| CGV | `#e74c3c` (빨강) |

### 뱃지 (무대인사/시사회)
| 유형 | 색상 |
|------|------|
| 무대인사 / 커튼콜 | `#c0392b` |
| 시사회 | `#1a6fa8` |
| 기타 | `#7f8c8d` |

- border-radius: `20px` (pill 형태)

### 예매 버튼
- 배경: `#1a1a2e` (다크 네이비, 모든 극장 공통)
- border-radius: `6px`
